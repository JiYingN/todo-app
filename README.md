# TaskFlow - 高效待办事项管理应用
TaskFlow 是一款基于原生 HTML、Tailwind CSS 和 JavaScript 开发的轻量级待办事项管理应用，专注于提供简洁、高效的任务管理体验，支持任务分类、优先级管理、截止日期设置等核心功能，所有数据存储在本地浏览器中，无需后端服务即可运行。

## ✨ 功能特点
| 功能 | 说明 |
|------|------|
| 任务全生命周期管理 | 支持任务的添加、编辑、删除、完成/取消完成操作 |
| 多维度分类管理 | 任务分类（BMS/NVH）+ 紧急程度分级（P0/P1/-） |
| 精准时间管控 | 截止日期时间设置，自动验证避免选择过去时间 |
| 智能筛选与搜索 | 支持按完成状态/分类筛选，关键词实时搜索任务 |
| 回收站机制 | 已删除任务保留30天，支持恢复/永久删除/清空操作 |
| 数据可视化 | 任务完成进度条、优先级分布统计，直观掌握任务状态 |
| 友好交互体验 | 操作反馈（Toast通知）、确认对话框、平滑动画效果 |
| 响应式设计 | 适配移动端、平板、桌面端等不同尺寸设备 |

## 🛠 技术栈
- **核心架构**：HTML5（语义化标签、表单验证）
- **样式引擎**：Tailwind CSS v3（自定义主题、工具类、动画）
- **交互逻辑**：原生 JavaScript（ES6+）
- **本地存储**：localStorage（数据持久化）
- **图标资源**：Font Awesome 4.7.0
- **布局适配**：响应式设计 + 弹性盒模型（Flex）

## 🚀 快速使用
### 运行方式
无需安装依赖，直接将 `todo-app.html` 文件拖入浏览器即可启动应用，支持 Chrome、Firefox、Edge 等现代浏览器。

### 核心操作指南
#### 1. 添加任务
- 填写**任务标题**（必填）、任务描述（可选）
- 选择任务分类（BMS/NVH）、紧急程度（P0/P1/-）
- 设置截止日期时间（必填，不可早于当前时间）
- 点击「添加任务」按钮完成创建

#### 2. 任务管理
- **编辑任务**：点击任务项右侧编辑图标，修改信息后保存
- **完成任务**：勾选任务左侧复选框，自动标记为已完成（划线样式）
- **删除任务**：点击任务项右侧删除图标，确认后移入回收站
- **恢复/删除任务**：切换至「回收站」标签，可恢复任务或永久删除

#### 3. 筛选与搜索
- **状态筛选**：点击「全部/未完成/已完成」按钮过滤任务
- **分类筛选**：点击「BMS/NVH」按钮筛选指定分类任务
- **关键词搜索**：在搜索框输入标题/描述关键词，实时过滤结果

#### 4. 批量操作
- 「清空已完成」：批量移除所有已完成的任务至回收站
- 「清空回收站」：永久删除回收站中所有任务（需确认）

## 🖥 界面说明
### 核心布局
1. **顶部区域**：应用标题 + 全局操作反馈（Toast通知）
2. **左侧面板**：
   - 任务表单（添加/编辑任务）
   - 任务概览统计（完成进度条、优先级分布）
3. **右侧面板**：
   - 搜索框 + 多维度筛选按钮
   - 任务列表（动态渲染，支持滚动加载）
4. **回收站页面**：已删除任务管理 + 保留规则提示

### 交互反馈
- 操作成功/失败：顶部 Toast 通知（3秒自动消失）
- 危险操作：弹出确认对话框（如删除/清空操作）
- 视觉动画：任务项滑入/淡入、模态框缩放、按钮hover效果

## 💾 本地存储说明
1. 任务数据（`tasks`）和回收站数据（`trash`）均存储在浏览器 `localStorage` 中
2. 数据持久化：刷新页面/重新打开浏览器后数据不丢失
3. 自动清理：回收站中超过30天的任务会在应用启动时自动清理
4. 数据隔离：仅在当前浏览器生效，更换设备/浏览器无法同步

## 🎨 自定义扩展
### 样式扩展
可通过修改 Tailwind 配置块扩展自定义样式：
```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                // 可新增/修改主题色
                primary: '#3B82F6',
                secondary: '#60A5FA',
                // ...
            },
            // 可新增字体、阴影、动画等
        }
    }
}
```

### 功能扩展
- 新增任务分类：修改 `<select id="taskCategory">` 中的 `option` 选项
- 调整优先级等级：修改 `<select id="taskPriority">` 中的 `option` 选项
- 自定义动画：修改 CSS 中的 `@keyframes` 或自定义工具类

## ⚠ 注意事项
1. 截止日期时间验证：系统会自动校验，不允许选择早于当前时间的日期
2. 数据安全：本地存储数据可通过浏览器开发者工具清除，建议重要任务做好备份
3. 兼容性：不支持 IE 浏览器，建议使用现代浏览器（Chrome ≥ 80、Firefox ≥ 75）
4. 文件类型说明：当前应用核心文件为 `todo-app.html`，若出现「当前不支持该文件类型，请尝试其他文件」报错，需确认打开的文件为 HTML 格式，且浏览器支持直接运行本地 HTML 文件
